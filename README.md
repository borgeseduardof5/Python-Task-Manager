# To-Do List em Python

Um simples gerenciador de tarefas em linha de comando (CLI) criado para estudos de lógica Python, estruturação de menus e manipulação de listas.

## Funcionalidades
- **Adicionar:** Insere novas tarefas na lista.
- **Mostrar:** Exibe todas as tarefas com um índice numerado.
- **Remover:** Remove uma tarefa específica com base no número.
- **Sair:** Encerra o programa.

## Como Executar
1. Certifique-se de ter o Python instalado.
2. Execute o script: `python lista_tarefas.py`

## Aprendizados (Nível Júnior)
- Uso de `while True` para loops infinitos de menu.
- Manipulação de listas (`.append()`, `.pop()`).
- Uso de `enumerate()` para indexação.
- Validação básica de entrada de dados.
